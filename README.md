KALI DApp is a decentralized application built on the Sonic Testnet blockchain that enables users to tokenize, trade, and manage real estate assets using smart contracts. It combines three main components:

RealEstateERC721: An ERC721-based contract for creating unique property tokens. Allows minting of tokens representing real estate properties with details like cadastral number, location, and valuation. 
RealEstateSale: A smart contract to manage the sale of real estate tokens. Facilitates setting buyers and sellers, completing sales, and handling payments securely. RentableToken: An ERC20-based token contract for a utility token called KLI. Enables users to buy, transfer, and burn tokens, as well as earn an annual return.
